# Leone Mangione (“Greedy Lion”)

A game made for and with my children using [Yeah](https://github.com/yeahrb/yeah#readme)

## Screenshot
![](http://inft.ly/PijFxsR.png)


## How to play

Press CTRL <kbd>⌃</kbd> to eat the zebra when its withing reach.


## Installation

```
bundle install
yeah serve
```
